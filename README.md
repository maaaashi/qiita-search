# Installation

```bash
$ git clone https://github.com/maaaashi/qiita-search.git
$ cd qiita-search
$ npm install
```

# Dev

```bash
$ npm run dev
```

# Usage

Please import the dist/ folder from the Google Chrome extensions menu.
